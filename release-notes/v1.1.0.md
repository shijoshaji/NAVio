### Smart Re-entry Logic
- **Re-entering Positions**: If you add a fund that you previously sold, the system now intelligently creates a **New Active Entry** instead of overwriting your historical sold record. This allows you to track multiple trade cycles for the same fund.

### Safety & Validation
- **Duplicate Prevention**: The "Add Fund" workflow now checks if the fund is already *active* in the selected group. If it exists, it prevents accidental duplication and prompts you to use the Edit function.

- **New Item Visibility**: Fixed an issue where newly added funds (with 0 investment) were hidden from the Dashboard. They are now always visible to allow for immediate editing.
